Design and Quality of Content

I used good design techniques and adherence to CRAP by using different sized headers and text to order and distinguish content. Relevant content and images are surrounded by a border making them pop out from the page. I have a lighter background with deeper and darker colors more upfront which helps content stand out even more. I use the same layout, spacing, and design for most of the pages, so users can expect where to find certain content. Related items are grouped together and or are within proximity with one another.

Content was made to be informative, less formal, and more personable with some of my own thoughts and experiences added in. I made sure to check over my spelling and grammar as well. Each page has pictures that are relevant to the content.

Quantity of Content

I have reached the requirement of 10 content pages with what I believe to be sufficient content but not overbearing. I also will have the grading page and references page adding up to 12 pages total. I think I added very pertinent information based off of which page it was.

Responsive Design

I designed the web page with mobile first in mind then put responsive media query rules to scale up for bigger device screens. Mobile first, then tablets, and then desktops. I wanted it to be a quality site for all users.

Miscellaneous Requirements

All of my code is properly indented to cleanly show hierarchy and what belongs to what. It is very organized and separated to avoid clutter and so that it can be easily read. I used the w3schools website to look up and research HTML and CSS tags to better achieve my design vision. Two of the tags used, which work in conjunction, were the details and summary tags to help me create a hierarchy nav.

I put some comments to label and minimally describe certain sections. No inline styles were used and there is only one global CSS file. Each page has a unique title so you can know which page you are on along with pictures in line with the title. Each page has a global nav and a Back to Top button. All HTML and CSS has passed validation. A reference page and this grading page detailing what I have done are also included.
